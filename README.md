# Ruang Seduh

Ruang Seduh adalah website cafe berbasis HTML, CSS, dan JavaScript. Website ini menampilkan informasi cafe, menu, promo, galeri, ulasan, reservasi, kontak, dan halaman Kelola dalam satu tampilan yang rapi dan mudah digunakan.

Tampilan website dibuat modern dengan nuansa cafe yang hangat. Setiap bagian dirancang agar pengunjung bisa melihat informasi dengan jelas, mulai dari menu makanan dan minuman, promo yang sedang berjalan, galeri suasana cafe, hingga kontak yang bisa langsung terhubung ke WhatsApp.

## Halaman Kelola

Halaman Kelola digunakan untuk mengatur isi website langsung dari browser. Data seperti nama cafe, tagline, deskripsi, warna tampilan, menu, promo, galeri, review, FAQ, kontak, dan pengaturan lainnya bisa diubah tanpa membuka file kode.

Setiap perubahan yang disimpan akan langsung digunakan oleh tampilan website.

## Data Website

Data awal website dibuat kosong. Tidak ada menu contoh, harga contoh, gambar bawaan, promo palsu, alamat palsu, nomor WhatsApp palsu, atau review palsu.

Isi website bisa ditambahkan sendiri melalui halaman Kelola.

## Penyimpanan Data

Website ini menggunakan LocalStorage untuk menyimpan data di browser. Data yang sudah diisi akan tetap tersimpan selama data browser tidak dihapus.

Jika website dibuka dari browser atau perangkat lain, isi datanya bisa berbeda karena penyimpanan dilakukan secara lokal di browser masing-masing.

Fitur Export Data bisa digunakan untuk menyimpan cadangan data. Fitur Import Data bisa digunakan untuk memakai kembali data yang sudah dicadangkan.

## Cara Menjalankan

Buka file `index.html` di browser.

Agar proses edit lebih nyaman, folder website bisa dibuka di Visual Studio Code lalu dijalankan menggunakan Live Server.
